Multi-Camera Vehicle Detection for Parking Management

The system has been designed so that existing parking lot security cameras can be used for the proposed system after a simple configuration, without the need for a complete new camera deployment.The designed system faces more complicated scenarios than ones tackled in state of the art: almost total occlusions and climatic changes (cloudy scenarios, rain, snow...), it's limits/reduces their performance. In this scenario with such a variable background it is not possible to carry out precise background extraction nor it's possible to label,define the region of each place as some parked vehicles completely occlude some of the spots behind them. In addition, the consideration of a scenario, which, as far as we know, has not been reported before for this type of systems, is added
